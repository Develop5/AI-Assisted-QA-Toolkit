# Incident Response Playbook (QA + DevOps)

## 1. Incident Stages
1. Detection  
2. Triage  
3. Mitigation  
4. Recovery  
5. Postmortem  

## 2. QA Responsibilities
- Reproduce incidents
- Validate fixes
- Provide logs and test evidence

## 3. DevOps Responsibilities
- Restore service
- Communicate status
- Document root cause

## 4. Postmortem Requirements
- Timeline
- Impact analysis
- Root cause
- Action items
- Prevention plan
