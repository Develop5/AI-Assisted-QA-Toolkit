# Load Testing Standards

## 1. Mandatory Metrics
- Response time
- Throughput
- Error rate
- CPU/memory usage

## 2. QA Responsibilities
- Define load profiles
- Validate system stability
- Validate degradation thresholds

## 3. DevOps Responsibilities
- Provide monitoring dashboards
- Ensure autoscaling works
- Validate infrastructure limits

## 4. Best Practices
- Use realistic traffic patterns
- Run load tests before major releases
