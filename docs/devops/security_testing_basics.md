# Security Testing Basics (QA + DevOps)

## 1. Mandatory Checks
- SQL injection
- XSS
- CSRF
- Authentication bypass
- Authorization gaps

## 2. DevOps Responsibilities
- Run SAST/DAST tools
- Scan dependencies
- Enforce secret scanning

## 3. QA Responsibilities
- Validate security flows manually
- Validate session expiration
- Validate password policies

## 4. Best Practices
- Use OWASP guidelines
- Rotate secrets regularly
