# GitOps Standards

## 1. Principles
- Declarative configuration
- Version-controlled infrastructure
- Automated reconciliation

## 2. QA Validation
- Validate environment consistency
- Validate configuration drift detection
- Validate rollback via Git revert

## 3. DevOps Requirements
- Use ArgoCD or Flux
- Use signed commits
- Use separate repos for infra & app

## 4. Best Practices
- Avoid manual changes in production
