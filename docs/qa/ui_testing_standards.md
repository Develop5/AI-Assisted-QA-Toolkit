# UI Testing Standards

## 1. Scope
Defines rules for UI automation and manual UI validation.

## 2. Mandatory Checks
- Layout consistency
- Accessibility compliance
- Input validation
- Error messages
- Navigation flow

## 3. Accessibility Rules
- All buttons must have labels
- Forms must support keyboard navigation
- Color contrast must meet WCAG AA

## 4. Browser Coverage
- Chrome (mandatory)
- Firefox (recommended)
- Safari (optional)
- Edge (mandatory for enterprise apps)

## 5. Best Practices
- Avoid brittle selectors
- Use page objects
- Capture screenshots on failure
