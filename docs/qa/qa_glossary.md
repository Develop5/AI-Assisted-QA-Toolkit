# QA Glossary

## Smoke Test
Minimal set of tests validating basic functionality.

## Regression Test
Tests ensuring existing functionality remains stable after changes.

## Flaky Test
A test that passes or fails inconsistently.

## Deterministic Test
A test that always produces the same result under the same conditions.

## Critical Path
Most important user flow in the system.

## Assertion
A validation step confirming expected behavior.

## Fixture
Reusable setup data or configuration for tests.

## Mock
Simulated version of a real dependency.

## Test Oracle
Source of truth for expected results.
