# DevSecOps Risk Matrix

## 1. Risk Levels
- Critical: immediate remediation
- High: fix before release
- Medium: fix next sprint
- Low: monitor only

## 2. Risk Factors
- Exploitability
- Impact
- Exposure
- Complexity
- Detectability

## 3. QA Responsibilities
- Validate risk classification
- Validate mitigations
- Validate regression tests for security fixes

## 4. Best Practices
- Reassess risk after every major change
