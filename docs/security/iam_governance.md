# Identity & Access Management (IAM) Governance

## 1. Principles
- Least privilege
- Role-based access control
- Zero trust

## 2. Mandatory Requirements
- MFA for admin accounts
- Password rotation every 180 days
- Logging of all access attempts

## 3. QA Responsibilities
- Validate role boundaries
- Validate privilege escalation protections
- Validate session termination

## 4. Best Practices
- Use centralized IAM
- Avoid shared accounts
